## Tools used

- [Bootstrap](https://getbootstrap.com)
- [Lorempixel](http://lorempixel.com) (for random image generation)

## Project setup

Clone the repository and install dependencies

```bash
$ git clone https://github.com/iamwebwiz/vue-jobber.git
$ cd vue-jobber
$ npm install
```

Run the project

```bash
$ npm run serve
```
